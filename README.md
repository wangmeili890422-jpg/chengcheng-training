# 🧒 诚诚周训练记录

迪爸英语周训练数据记录与管理工具 —— 上传打卡截图，AI 自动识别，一键填入统计表。

> 全程浏览器本地运行，无需安装任何软件，数据不上传服务器。

---

## 📖 如何使用

### 第一步：获取工具

点击本页面的 `index.html` → 右上角点 **「下载」** 按钮，保存到本地。

### 第二步：双击打开

直接 **双击** `index.html`，浏览器会自动打开工具界面。

### 第三步：操作流程

#### 方式一：AI 自动填写（推荐 ✨）

1. 点击 **「🤖 自动填写」** 按钮
2. 上传迪爸小程序的打卡截图（支持拖拽）
3. 工具自动 OCR 识别图片中的文字内容
4. 智能解析磨眼睛、磨耳朵、闪卡、外教等各项数据
5. 自动填入对应的周训练统计表中

> 识别失败？可以手动粘贴打卡文字，同样自动解析！

#### 方式二：手动编辑

1. 点击统计表中的任意单元格
2. 在弹窗中输入内容及时长
3. 点击保存即可

## ✨ 功能特点

### 🤖 AI 智能识别
- 上传打卡截图，自动 OCR 识别文字内容
- 智能解析磨眼睛/磨耳朵/闪卡/外教等 8 个训练项目
- 自动识别文件名所属系列（动物兄弟、字母积木、数字积木、螺丝钉等）
- 30 秒超时保护，失败自动切换手动模式

### 📋 周训练统计表
- 一周 7 天 × 8 个训练项目的完整记录
- 支持内容 + 时长双维度记录
- 自动计算每日合计、周汇总
- 周末行黄色标记，一目了然
- 所有单元格可点击编辑

### 📊 数据趋势图表
- **磨眼睛时长趋势** — 柱状图 + 累计/日均摘要
- **磨耳朵时长趋势** — 柱状图 + 累计/日均摘要
- **各项目时长占比** — 环形图，中心显示总分钟
- **每日总时长趋势** — 柱状图

### ☁️ 云端同步
- 支持邮箱登录 / GitHub 登录
- 数据自动同步到云端，多设备无缝切换
- 登录后自动加载云端数据

### 📥 导入导出
- **导出 Excel** — 一键导出 CSV 格式
- **备份数据** — JSON 全量备份
- **导入备份** — 恢复数据，兼容旧格式

### 其他功能
- ↩️ **撤销** — 恢复上一状态
- 🗑️ **彻底清零** — 清空当前周数据
- ➕ **新一周** — 自动生成下一周
- 🔗 **分享** — 复制当前页面链接
- 📅 **周切换** — 自由切换不同周的数据
